Radix is a client-side framework for building model-driven decentralized
applications on top of Blockstack storage and authentication.
It aims to solve the following problems:

  • Storing and querying complex data
  • Indexing data to easily query user's publicly saved data
  • Collaboration with other users
  • Real-time updates (in progress)
